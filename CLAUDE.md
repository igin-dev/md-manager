# md-manager — система управления MD-файлами для Claude Code

Пакет из 5 slash-команд и 3 субагентов для полного цикла работы с CLAUDE.md файлами.

## Структура проекта

```
.claude/
  commands/          # slash-команды (точки входа для пользователя)
    md-init.md       # создать новый MD-файл
    md-audit.md      # проверить по чеклисту
    md-fix.md        # исправить найденные проблемы
    md-sync.md       # обновить после изменений в проекте
    md-optimize.md   # оптимизировать разросшийся файл
  agents/            # субагенты (специализированная логика)
    md-creator.md    # создаёт новые MD-файлы
    md-auditor.md    # проверяет по встроенному чеклисту
    md-optimizer.md  # планирует оптимизацию
    rules/           # вспомогательные правила (читаются md-creator через Read)
    templates/       # шаблоны для новых файлов
    examples/        # реальные примеры хороших MD-файлов
templates/           # копии шаблонов для удобного просмотра
install.ps1          # установка на Windows
install.sh           # установка на Linux/macOS
```

## Установка в целевой проект

```powershell
# Windows
.\install.ps1 C:\path\to\your\project

# С перезаписью существующих файлов
.\install.ps1 C:\path\to\your\project --force
```

## Рабочий процесс

```
# Новый проект
/md-init → создаёт CLAUDE.md из контекста проекта

# Проверка и исправление
/md-audit [файл] → отчёт по чеклисту
/md-fix [файл]   → исправляет найденные проблемы

# Поддержка актуальности
/md-sync [описание изменения] → обновляет после изменений в проекте

# Оптимизация
/md-optimize [файл] → предлагает план разбивки разросшегося файла
```

## Архитектура субагентов

Субагенты в `.claude/agents/` вызываются через Agent tool из slash-команд.
Они не вызываются пользователем напрямую — в description нет триггерных слов.

- `md-auditor`: model=sonnet, tools=[Read, Glob] — чеклист встроен в промпт (source of truth для кодов)
- `md-creator`: model=sonnet, tools=[Read, Glob] — читает rules/ и examples/ через Read
- `md-optimizer`: model=sonnet, tools=[Read, Glob] — правила встроены в промпт

Rules/ файлы — справочник для md-creator и людей, не для аудитора.
Коды аудита (RED-1, NAV-1, EXT-1 и т.д.) определены только в md-auditor.md.

## Запреты

- Агент никогда не записывает файлы без явного подтверждения пользователя ("да")
- Субагенты не вызывают другие субагенты — только главная slash-команда делает это
